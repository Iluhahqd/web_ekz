# Электронная библиотека

Работа: Невретдинов Илья Рушанович, группа 241-327.

Реализован вариант 2: пользователи с ролью «Пользователь» могут создавать подборки книг, просматривать свои подборки и добавлять книги в выбранную подборку.

## Запуск

```bash
python3 -m venv .venv
.venv/bin/python -m pip install -r requirements.txt
.venv/bin/python app.py
```

После запуска приложение будет доступно по адресу `http://127.0.0.1:5000`.

Тестовые пользователи:

- `admin` / `admin`
- `moderator` / `moderator`
- `user` / `user`
